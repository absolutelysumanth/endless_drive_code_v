🚗 Endless Coastal Drive

A smooth browser-based 3D driving game built with Three.js and a custom lightweight engine.

Drive endlessly through multiple environments with realistic controls, fuel management, and optimized performance.

🎮 Features
⚡ Smooth gameplay (fixed 60 FPS simulation + interpolation)
🌍 Endless procedural world
🏙️ Multiple zones:
City
Forest
Beach Highway
Desert Canyon
Snow Mountains
Neon Valley
🚘 Manual gear system (R → 5)
⛽ Fuel system with petrol bunks
🚦 Car features (lights, horn, camera modes)
🗺️ Mini-map + full map navigation
🌗 Day / Night cycle
🚗 AI traffic system
🕹️ Controls
Action	Key
Move Forward	W / ↑
Move Backward	S / ↓
Turn Left	A / ←
Turn Right	D / →
Upshift	Shift
Downshift	Ctrl
Handbrake	Space
Headlights	H
Horn	G
Change Camera	C
Open Map	M
Refuel	R
Pause	Esc
🛠️ Tech Stack
Three.js
Vanilla JavaScript
HTML + CSS
