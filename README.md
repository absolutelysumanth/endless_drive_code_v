Overview

Endless Coastal Drive is a smooth browser-based 3D driving game built using Three.js and a custom lightweight game engine.
The game features an endless road, multiple environments, realistic driving mechanics, and performance optimizations for a seamless experience.

Key Features
⚡ Smooth Gameplay
Fixed 60 FPS simulation
Interpolated rendering for smooth visuals
Optimized performance with adaptive quality
🌍 Dynamic World
Procedural endless road generation
Multiple environments (zones):
City
Forest
Beach Highway
Desert Canyon
Snow Mountains
Neon Valley
🚘 Driving System
Manual gear shifting (Reverse to Gear 5)
Realistic acceleration and braking
Steering with sensitivity control
Handbrake support
⛽ Fuel System
Limited fuel capacity
Petrol bunk stations along the road
Refuel by stopping near stations
🚦 Car Features
Headlights toggle
Horn system
Multiple camera modes
🗺️ Navigation System
Mini-map and full map view
Distance tracking
Nearest fuel station indicator
🌗 Day/Night System
Day mode
Night mode
Automatic time cycle
🚗 Traffic System
AI-controlled vehicles
Continuous traffic flow
Controls

Move Forward: W or Up Arrow
Move Backward: S or Down Arrow
Turn Left: A or Left Arrow
Turn Right: D or Right Arrow
Upshift Gear: Shift
Downshift Gear: Ctrl
Handbrake: Space
Headlights: H
Horn: G
Change Camera: C
Open Map: M
Refuel: R
Pause Game: Esc

Project Structure

index.html – Main entry point

How to Run
Download or clone the repository:

Open the project folder
Run the game by opening:
